# Unity Vault — Free Resources

A curated collection of free design resources built for the ERLC community.

Every file in this repository is free to download and use. No paywalls. No sign-ups. No restrictions.

---

## What Is Unity Vault

Unity Vault is a curated resource vault built for ERLC server owners, designers, and communities. The goal is simple: organize the tools and resources that successful servers use and make them accessible to everyone.

---

## What Is Included

This repository contains free downloadable resources including:

- Photoshop artboard templates for ERLC server design
- Adobe Illustrator artboard templates with size guides
- Discord server structure templates
- PDF guides and references

New resources are added over time. Watch or star this repository to stay updated.

---

## How to Use

1. Go to the Releases tab
2. Find the resource you need
3. Click the file to download it
4. Open it in the appropriate software and start building
